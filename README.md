# pioneer_research
Hiromi's repository to track research progress.

Predicting housing prices is an important problem for buyers, sellers, and policymakers because housing affordability affects millions of people and influences decisions about where to live, work, and invest. Data science can be used to identify which factors have the strongest impact on property values, such as floor area, age of the building, distance to the nearest station, and location. Machine learning models could be trained to predict housing prices and compare the accuracy of different approaches.
